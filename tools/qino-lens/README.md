# qino-lens

The Qualities of Attention. A library of **Lenses** to perceive text through specific sensitivities.

## Commands

### `qino-lens:wanderer`

Views input text through the Wanderer's Lens (Sensitivity to Friction).

- **Usage:** Pipe technical summaries or logs to see them through the lens of consequence.
- **Principles:** Contextual Density, Indirect Action.
