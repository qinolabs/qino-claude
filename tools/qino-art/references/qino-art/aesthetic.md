# qino-art Visual Aesthetic

**The definitive style guide for all qino-art visual content.**

---

## Core Identity

**Tron-meets-Excalidraw** - Digital warmth combining:
- Hand-drawn Excalidraw-style sketch lines (NOT clean vectors)
- Dark slate backgrounds for modern contrast
- Neon orange (warmth) + cyan (tech) accents
- Subtle glows on key elements

---

## Style References

- **Saul Steinberg** - Playful line quality, conceptual wit
- **New Yorker covers** - Editorial sophistication, single compelling image
- **Risograph prints** - Flat colors, slight misregistration, tactile quality
- **Matisse cutouts** - Bold shapes, decisive composition

---

## Color System

### Backgrounds

| Name | Hex | Usage |
|------|-----|-------|
| White | #FFFFFF | For transparency removal (use with remove.bg) |
| Cream | #FAF3E0 | Warm, inviting |
| Terracotta | #E07A5F | Earthy, grounded |
| Sage | #81B29A | Calm, natural |
| Peach | #F4A261 | Friendly, approachable |
| Slate | #2D3436 | Modern, dark mode |

**Rule:** Single flat color only. NO gradients ever.

### Primary: Black Linework

- **Hex:** #000000
- **Role:** DOMINANT - carries all composition
- **Quality:** Hand-drawn, variable weight, imperfect, gestural

### Accent Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Deep Purple | #4A148C | Brand accent - REQUIRED but subtle |
| Deep Teal | #00796B | Secondary accent option |

**Rule:** Color is ACCENT only. Black linework dominates. Purple/Teal should be visible and noticeable, not microscopic.

---

## Linework Rules

1. **Variable stroke weight** - Thicker where pressure would be
2. **Imperfect lines** - Slightly wobbly, human quality
3. **Gestural strokes** - NOT smooth vectors
4. **Hand-drawn quality** - Must look sketched, not computer-perfect

---

## Composition Rules

1. **2-3 elements maximum** - Simplicity is key
2. **40-60% element fill** - Elements should occupy this much of frame
3. **30-40% negative space** - Breathing room
4. **Full-bleed preferred** - Elements extend to or bleed off edges
5. **Square aspect ratio** - 1:1 format default

---

## Forbidden Elements

- Gradients (anywhere)
- Shadows or glows
- 3D rendering or depth
- Glossy/shiny surfaces
- Smooth perfect vector lines
- Saturated or cool colors (except accent)
- Photorealistic elements
- Multiple panels or collage layouts

---

## Signature

Small "{{{assistantName}}}" in charcoal (#2D2D2D) bottom right corner. No other text.

---

## Validation Checklist

Before any image is complete:

- [ ] Flat solid background (single color, no gradient)
- [ ] Black hand-drawn linework (imperfect, gestural)
- [ ] Abstract conceptual metaphor (not literal)
- [ ] Full-bleed or well-composed with negative space
- [ ] Square 1:1 aspect ratio
- [ ] 2-3 elements maximum
- [ ] No gradients, shadows, or shine
- [ ] Color visible - Purple/Teal noticeable (not microscopic)
- [ ] Signature present bottom right
